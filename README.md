# Vue.js Frontend App

This is the frontend application for the After School Activities coursework.

## Links

- **Live Website (GitHub Pages):** https://neluantony.github.io/coursework-frontend/
- **GitHub Repository:** https://github.com/neluantony/coursework-backend
- **Backend Server:** https://coursework-backend-qzv7.onrender.com

## Features

- Vue.js 3 with Composition API.
- Real-time search and sorting.
- Shopping cart functionality.
- Order checkout validation.
